# JavaScript Loops Practice

This repository contains my JavaScript practice code for learning different types of loops and array methods. I created simple examples using objects and arrays to understand how each loop works and when to use it.

## Topics Covered

- `for` loop
- `forEach()`
- `for...in` loop
- `for...of` loop
- `map()`
- `filter()`
- Objects
- Arrays

## What I Learned

- Iterating through arrays using different loop types.
- Accessing object keys and values.
- Understanding the differences between `for`, `forEach`, `for...in`, and `for...of`.
- Using `map()` to create a new array.
- Using `filter()` to return elements based on a condition.
- Choosing the right loop for different situations.
